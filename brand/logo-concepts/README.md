# Agent Ferry Logo Concepts

> 状态：Exploration
> 日期：2026-07-20
> 范围：第一轮图形方向，不代表已经采用的正式品牌资产

本轮围绕“把完整上下文可靠地交给一个明确的 Agent”设计。所有 SVG 使用 `viewBox="0 0 100 100"` 和 `currentColor`，优先验证 Chrome 工具栏、扩展标题栏、CLI 与项目头像等小尺寸场景。

## 方向

1. **Ferry Gate**：双港口与中央载荷，直接表达跨边界交接；当前首选。
2. **Ferry Monogram**：模块化 `F` 与独立载荷，品牌识别最强。
3. **Context Capsule**：代码括号与上下文点阵，开发者工具属性最明显。
4. **Converging Wake**：多路内容汇聚为一次交接，动势最强。
5. **Folded Handoff**：页面折角进入 Agent 通道，网页捕获语义最直接。
6. **Dock Network**：本地中枢与可选目标，最贴近当前系统拓扑。

## 评审顺序

先打开 [showcase.html](./showcase.html)，按以下顺序判断：

1. 只看 16px 与 28px，排除轮廓不清楚的方向。
2. 切换黑色、蓝色和橙色，确认图形不依赖特定颜色成立。
3. 切换明暗背景，检查负空间与笔画重量。
4. 最后再考虑品牌含义，选择 2–3 个方向进入精修。

## 第一轮约束

- 不使用机器人、脑、AI 星芒或聊天气泡。
- 不使用具象船、锚和海浪，避免航运或旅游品牌联想。
- 不依赖渐变、阴影、透明度或外部字体。
- 动效只用于展示页，正式静态资产必须独立成立。
