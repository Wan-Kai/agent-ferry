说明
----
Agent Ferry 将你正在阅读的网页交给你选择的 AI Agent。

使用前先确认当前页面，选择 Agent 及其运行位置，检查完整任务指令，然后开始交接。你可以在本地任务历史中查看正在运行和已完成的任务。

• 仅当你点击“开始”后，才提取当前文章、X/Twitter 帖子或支持的 arXiv 页面
• 可将内容发送到指定工作目录中的本地 Claude Code、Codex 或 OpenCode
• 可将内容发送到你自行配置的远程 Hermes
• 在本地保存可复用的 Prompt 模板和有上限的任务历史
• 使用 activeTab 临时权限，不申请对所有网站的永久访问

Agent Ferry 需要安装轻量的 macOS 端 Agent Ferry Core。它不会安装或替代 Claude Code、Codex、OpenCode 或 Hermes。所选 Agent 及其模型服务配置由你负责。
