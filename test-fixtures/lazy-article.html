Agent Ferry 懒加载长文验证

这篇文章用于验证浏览器扩展能够提取普通网页、保留来源元数据，并把完整 Markdown 连同用户可见的 Prompt 一次性交给 Hermes。

第一部分描述内容捕获。扩展会在有界次数内滚动页面，让合理的懒加载正文出现；无论成功或失败，最终都必须恢复用户原来的阅读位置。

第二部分描述安全边界。浏览器只能选择已经由守护进程发现并诊断为可用的单个目标，Bearer Token 始终留在 macOS Keychain 与守护进程边界内。

第三部分描述任务生命周期。Hermes 接收 Run 后拥有远端任务，浏览器弹窗只订阅实时事件；关闭弹窗不会触发 stop，也不会取消任务。
